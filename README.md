# Nexora

Sistema de gestão desenvolvido em React para gerenciamento de clientes, pedidos, produtos e orçamentos.

## Funcionalidades

- Autenticação de usuários
- Dashboard com indicadores
- Gestão de clientes
- Gestão de pedidos
- Gestão de produtos
- Gestão de orçamentos
- Relatórios gerenciais

## Tecnologias utilizadas

- React
- React Router DOM
- JavaScript
- CSS
- LocalStorage

## Objetivo

Projeto desenvolvido para estudo e aprimoramento de habilidades em desenvolvimento front-end, focando em componentes reutilizáveis, gerenciamento de estado, rotas protegidas e construção de interfaces modernas.

## Próximas implementações

- Backend em Python
- Integração com banco de dados
- Controle de usuários
- Relatórios avançados
- API REST

## Como executar

```bash
npm install
npm run dev
```
